# Bionic Reader - ADHD Reading Assistant

A lightweight React + Vite web app for Bionic-style reading, enhancing focus by bolding the first part of English words.

## Features

- Paste any English text and see it transformed with Bionic-style formatting
- Adjustable bold ratio (0.2 - 0.8)
- Option to ignore short words (â‰¤3 characters)
- Customizable font size and line height
- Copy rich text to clipboard (preserves bold formatting)
- Dark mode support

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- Vite
- React (JavaScript)
- Pure CSS (no UI libraries)
